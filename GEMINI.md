# Terraqua Company Profile

A modern, responsive company profile website for **Terraqua (PT. Terraqua Alam Nusantara)**, a sustainable shrimp farming company based in Belitung, Indonesia.

## Tech Stack

- **React 19** + **Vite**: Fast frontend build tool and framework.
- **Tailwind CSS v4**: Utility-first CSS framework for styling.
- **Framer Motion**: Library for production-ready animations (scroll reveals, transitions).
- **React Router DOM**: Client-side routing for SPA navigation.
- **Lucide React**: Beautiful & consistent icons.

## Project Structure

```
src/
├── components/     # Global UI components (Navbar, Footer, Layout)
├── pages/          # Page views (Home, About, WhatWeDo, Expansion, Contact)
├── assets/         # Static assets
├── App.jsx         # Main application component with Routing
└── main.jsx        # Entry point
```

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Start the development server:**

    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Interactive Elements**: Animated statistics, tabs for expansion plans, and smooth scroll reveals.
- **Modern UI**: Clean "Clean Marine" aesthetic with deep blues and cyan accents.
